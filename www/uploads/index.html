Upload

Multipart / Form-Data

Multiple files bundled in one request, separated by a boundary. Each part has its own Content-Disposition and Content-Type.

Raw Bytes — File

File content sent directly as the body with Content-Type: application/octet-stream. No boundaries, no field names.

Raw Bytes — Text

Plain text sent directly as the body with Content-Type: text/plain. The filename is passed via the X-Filename header.
